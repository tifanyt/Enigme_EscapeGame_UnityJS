Durant le DUT MMI, avec 5 autres étudiants, nous avons créé un escape game avec des énigmes numériques. Un de mes rôles dans ce projet était de coder certaines énigmes, pour cela j'ai utiliser Unity en javascript. 
La première énigme consiste à retrouver dans la pièce les lignes de codes fictives qu'ils doivent ensuite rentrer sur l'ordinateur.
La seconde énigme est un mastermind : grâce à des énigmes faites au préalable, les joueurs vont trouver plusieurs groupes de 3 chiffres ou lettres. Grâce au mastermind, ils doivent mettre ces groupes dans l'ordre pour retrouver une suite complète.
